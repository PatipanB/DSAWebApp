# DSA Visualizer

Interactive visualizations of data structures and algorithms with play/pause/step controls, real-time code highlighting, Big-O badges, and mapped NeetCode practice problems.

## Stack

Vite · React · TypeScript (strict) · Tailwind CSS v3 · Framer Motion · Zustand · React Router · Vitest · pnpm.

## Develop

```bash
pnpm install
pnpm dev
```

## Verify

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
